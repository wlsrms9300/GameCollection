## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

실행 후 '시작하기' 버튼으로 테트리스 게임 진행가능

- 완료

  1. 테트리스 게임
  2. 배경음악 추가

- 개발예정

  1. 회원가입 및 로그인 (디자인 완료)
  2. supabase연동 (완료)
  3. supabase에 database구성 (테이블 구성중)
  4. 점수 랭킹보드 추가
